Help

The CREW (Collaborative Research Events on the Web) web application improves access to research event content by capturing and publishing the scholarly communication that occurs at events like conferences and workshops.

In this current version of the software you can browse events via the facets that appear in the left hand column of the user interface. You can browse events by title, subject-area, location and date. You will be able to drill further into the event details to see information on individual sessions. If you see the icon you will be able too see a video of the presentation.
